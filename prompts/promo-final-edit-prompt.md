# Final Promo Image Edit - Text Overlay + Crop

## Nano Banana Editing Prompt

**Input Image:** `Gemini Generated Image (5).png`

**Task:** Add typography and crop to square format

### Prompt:
```
Add text overlay to this image: "DOTPRODUKT" in large geometric sans-serif
font (Eurostile or Bank Gothic style), all caps, tight kerning, positioned
at bottom of image, white text with subtle black outline or drop shadow for
contrast, professional band logo treatment, minimal and technical aesthetic,
preserve all existing image quality and lighting. Then crop to square 1:1
aspect ratio, centered on both figures, maintaining the lighting contrast
between left (cold blue) and right (warm amber) sides.
```

### Alternative Placements:

**Top Placement:**
```
Add text overlay to this image: "DOTPRODUKT" in large geometric sans-serif
font (Eurostile or Bank Gothic style), all caps, tight kerning, positioned
at top center of image, white text with black outline for readability,
professional band logo, technical minimal aesthetic. Crop to square 1:1
aspect ratio centered on both figures.
```

**Bottom with Subtitle:**
```
Add text overlay: "DOTPRODUKT" in large geometric sans-serif font at bottom
center, with "—SLOPMASCHINE" in smaller matching font below it, all caps,

```

## Specs

**Text:**
- Primary: `DOTPRODUKT` (no umlauts, all caps)
- Optional subtitle: `—SLOPMASCHINE` or `SLOPMASCHINE`

**Font Style:**
- Geometric sans-serif
- Eurostile, Bank Gothic, DIN, or similar
- Tight letter spacing
- Technical/industrial feel

**Text Treatment:**
- Color: White (#FFFFFF)
- Outline/Shadow: Black or dark gray for contrast
- Size: Large enough to be readable but not overwhelming
- Positioning: Bottom center (recommended) or top center

**Crop:**
- Aspect ratio: 1:1 (square)
- Center on both characters
- Maintain cold blue (left) and warm amber (right) lighting split
- Keep both faces fully visible

**Output Use:**
- Instagram post (1080x1080)
- Album art placeholder
- Square social media posts

## If Text Overlay Doesn't Work in Nano Banana:

Export the cropped square image without text, then add typography in:
- Photoshop
- Figma
- Canva
- Any image editor with text tools

### Font Recommendations (Free):
- **Eurostile** (if available)
- **DIN** (Google Fonts has alternatives)
- **Orbitron** (free, similar geometric style)
- **Rajdhani** (free, tight geometric)
- **Michroma** (free, very geometric and technical)

### Manual Text Settings:
- Font size: ~80-120pt (depending on canvas size)
- Letter spacing: -50 to -100 (tight)
- Color: White #FFFFFF
- Stroke: 2-3px black
- Position: Bottom, centered, ~50-100px from bottom edge
